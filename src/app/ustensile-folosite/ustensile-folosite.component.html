Sunt de 3 tipuri:

  • Stangaci

Capetele de freză sunt confecționate din 4 tipuri de materiale:
  1. Carbid
  2. Titan
  3. Ceramica
  4. Diamant

Culoarea bitului ne indică duritatea.

Negru – Extra Dur / Grosier

Verde – Dur / Grosier

Albastru – Mediu / Standard

Mov – Mediu spre Fin / Standard

Roșu – Fin

Galben – ExtraFin

Ustensile Unghii

✓ Pile unghii – sunt compuse din miez de PVC / ABS / LEMN / FIER, acoperite cu grit. Granulația pilei este dată de densitatea Grit-ului pe cm2. În funcție de granulația pe care o are fiecare pilă, se utilizează pentru :

  • 80 – 150 pentru îndepărtarea / pilirea materialului vechi
  • 150 – 180 pentru uniformizarea materialului
  • 240 pentru pregătirea plăcii unghiale, forma / scurtarea unghiei etc.

✓ Buffer / pilă buffer

  • Buffer – confecționat din burete abraziv
  • Pilă Buffer – miez din PVC / ABS, acoperite cu burete abraziv.
